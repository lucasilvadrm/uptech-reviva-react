## :checkered_flag: Starting

bash
# Clone this project
```sh
$ git clone https://github.com/lucasilvadrm/uptech-reviva-react-next
```

# Access
```sh
$ cd uptech-reviva-react-next
$ npm install
$ npm run build
```

# Open in your favorite browser
